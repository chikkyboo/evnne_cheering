🛠 カレンダーの編集方法（Voteチーム専用）

このファイルは calendar/schedule.html です。

編集方法：

  1. GitHub のリポジトリを開く
  2. calendar/schedule.html をクリック
  3. 右上の ✏（鉛筆）アイコンを押す
  4. .day-card をコピーして日付を追加
  5. 事務所 → class="item agency"
  6. Vote → class="item vote"
  7. ページ下部の「Commit changes」で保存

※ HTMLタグは .day-card の複製だけでOK。
※ デザインは style.css が担当するので壊れません。
