CDショップ情報の編集方法

Voteチーム専用マニュアル

📁 編集するファイル

CDショップ情報は以下のファイルで管理されています：

cdshops/list.html

📝 編集手順

  1. GitHub のリポジトリを開く
  2. cdshops/list.html をクリック
  3. 右上の ✏（鉛筆）アイコンを押す
  4. 該当する「形態」の下に .cdshop-card を追加
  5. 特典・価格・リンクを更新
  6. 「Commit changes」で保存

💡 編集のコツ

  • 形態（通常盤 / デジパ / 5枚セット / FC）ごとに整理すると見やすい
  • 同じ店舗で複数形態がある場合はカードを複製
  • 特典はショップごとに異なるので必ず記載
  • デザインは style.css が担当 → 壊れません

📦 カードのテンプレ

新しいショップを追加するときは、以下をコピーして使ってください：

<div class="cdshop-card">
  <h4>📍 ショップ名</h4>
  <ul>
    <li>特典：TBD</li>
    <li>価格：TBD</li>
    <li>販売開始：TBD</li>
  </ul>
  <a class="btn" href="#" target="_blank">ショップを開く</a>
</div>
